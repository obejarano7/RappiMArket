# RappiMArket
CarShop
- EL CARRITO DE COMPRAS ESTA CONSTRUIDO EN SU TOTALIDAD CON HTML, JQUERY, JAVASCRIPT Y BOOTSTRAP. SE UTILIZA UNAS LINEAS DE PHP PARA OBTENER LOS DATOS DEL ARCHIVO .JSON
- SE PUEDE OBTENER LA INFORMACIÓN MEDIANTE UNA RUTA WEB, INDICANDO EL NOMBRE Y EXTENSIÓN DEL ARCHIVO, EJEMPLO http://jsonfiles.azurewebsites.net/data1.html
- LA TABLA UTILIZADA PARA EL CARRITO SE CONTRUYO UTILIZANDO UN PLUGIN DE BOOTSTRAP LLAMADO BOOTSTRAP-TABLE Y SUS RESPECTIVAS EXTENSIONES (SE HICIERON MODIFICACIONES PARA CUMPLIR CON LOS REQUERIMIENTOS)
- PARA LA PERSISTENCIA DE LOS ITEMS SE UTILIZO EL SESSIONSTORAGE  EL CUAL MANTIENE LOS DATOS DEL .JSON EJECUTADO MIENTRAS NO SE CIERRE EL NEVEGADOR O SE CAMBIE LA URL DEL .JSON

* RECOMIENDO EJECUTAR EL CARRITO EN UN SERVER LOCAL COMO APACHE, EN MI CASO UTLICE XAMPP PARA INSTALAR APACHE EN MI EQUIPO. 
LA RAZÓN ES QUE HAY UN ARCHIVO PHP QUE ES QUIEN OBTIENE LA INFORMACIÓN JSON Y LA RETORNA AL NAVEGADOR

* PARA SU COMODIDAD HE ALOJADO LA APLICACIÓN EN LA RUTA  http://rappimarket.azurewebsites.net/ LA CUAL CARGARA POR DEFECTO CADA VEZ QUE SE ABRA NUEVA VENTANA DEL BROWSER 
LA INFORMACIÓN JSON DE LA RUTA http://jsonfiles.azurewebsites.net/data1.html.  	

* QUEDO ATENTO A CUALQUIER DUDA SOBRE EL FUNCIONAMIENTO DEL CARRITO, Y DE ANTEMANO DISCULPAS SI DURANTE LA EXPEREINCIA SE ENCENTRAN ALGUNOS BUGS.
* ESPERO QUE ESTE TRABAJO SEA DE SU AGRADO YA QUE TOMO BASTANTE ESFUERZO Y TIEMPO EXTRA EL REALIZARLO Y ASI CUIDAR LO MEJOR POSIBLE DE LOS DETALLES DEL MISMO.
